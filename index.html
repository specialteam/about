درباره من :

توسعه‌دهنده وب با بیش از ۸ سال تجربه در زبان‌های برنامه‌نویسی متنوع شامل Python، PHP، و JavaScript. متخصص فول استک در سطح میدلول با توانمندی‌های گسترده در طراحی و پیاده‌سازی سامانه‌های پیچیده و مدیریت پروژه‌های فناوری اطلاعات.
مهارت‌های فنی شامل:
- برنامه‌نویسی: تخصص در Python (80%)، FastAPI (60%)، Django (50%)، و PHP (60%).
- تحلیل داده‌ها: تجربه در تحلیل داده‌ها با استفاده از ابزارهای پیشرفته و توانمندی در استخراج و پردازش داده‌ها از منابع مختلف.
- اتوماسیون: مهارت در اتوماسیون شبکه‌های اجتماعی نظیر Telegram Bot API (90%) و Telegram Bot CLI (90%)، همچنین توانایی در اتوماسیون پلتفرم‌هایی مانند Instagram، Rubika و Eita.
- توسعه وب: تخصص در HTML (60%)، CSS (60%)، Bootstrap v5 (70%)، Tailwind CSS (40%)، و Vue.js (40%).

تجربه کاری شامل:
- طراحی و پیاده‌سازی سامانه لایسنس برای مدیریت مشتریان که با بهره‌گیری از آخرین تکنولوژی‌های روز به بهبود فرآیندهای کسب و کار کمک کرده است.
- توسعه سامانه اتوماسیون بر بستر فایربیس که به افزایش کارایی و کاهش هزینه‌های عملیاتی انجامیده است.
- پیاده‌سازی سامانه پایش و رصد بورس ایران که با استفاده از تحلیل داده‌ها و الگوریتم‌های پیشرفته به تحلیلگران بازار کمک می‌کند.
- طراحی و اجرای سامانه تبلیغات در بستر فضای مجازی و سامانه افزایش بازدید در شبکه‌های اجتماعی که به کسب و کارها در بهبود حضور آنلاینشان کمک می‌کند.
- توسعه سامانه استخراج اطلاعات بر بستر بلاکچین و رصد فضای بلاکچین برای تحلیل و استفاده از داده‌های بلاکچین.
- پیاده‌سازی سامانه رصد وضعیت جاری کریپتو که به کاربران در پیگیری و تحلیل ارزهای دیجیتال کمک می‌کند.
- طراحی سامانه ضد تحریم و تحریم‌شکن که به کاربران ایرانی امکان دسترسی به سرویس‌های بین‌المللی را فراهم می‌کند.

پروژه‌های اپن سورس شامل:
- CryptoScraper - پروژه‌ای برای استخراج اطلاعات ارزهای دیجیتال (2016)
- TelegramScraper - پروژه‌ای برای استخراج اطلاعات از تلگرام (2016)
- GoWooCommerceAPI - پروژه‌ای برای ارتباط با WooCommerce API با زبان Go (2016)
- PyWooCommerce - پروژه‌ای برای ارتباط با WooCommerce API با زبان Python (2016)
- greed_and_fear - پروژه‌ای برای تحلیل احساسات بازار (2016)
- xobot - پروژه‌ای برای اتوماسیون در تلگرام (2016)
- pymediabot - پروژه‌ای برای اتوماسیون رسانه‌های اجتماعی (2016)
- DontEdit - پروژه‌ای برای مدیریت و عدم تغییر داده‌ها (2016)
- systeminfo - پروژه‌ای برای استخراج اطلاعات سیستم (2017)
- dostet_beshnas - پروژه‌ای برای شناسایی دوستان در شبکه‌های اجتماعی (2017)

کار با ابزارها و فریمورک‌های مختلف نظیر Django، FastAPI، Vue.js و Tailwind CSS، همچنین توانایی بالا در مدیریت پروژه‌های فناوری اطلاعات و همکاری موثر با تیم‌های چند تخصصی. تسلط بر سیستم عامل‌های Debian Base (Linux) و مهارت‌های Lpic.
